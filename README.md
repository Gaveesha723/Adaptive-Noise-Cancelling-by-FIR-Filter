# Adaptive-Noise-Cancelling-by-FIR-Filter
Adaptive noise cancelling by FIR filter using a reference noise signal.
This repository implements an Adaptive FIR filter designed to extract clean audio (like speech) from a noisy signal. This method relies on an "adaptive linear element" and a Reference Signal approach. By providing the system with a separate recording of the background noise environment, the algorithm intelligently learns the noise profile and subtracts it from the primary audio, leaving the clean target signal behind
